# W2O SALADA - 샐러드 & 새벽배송 웹사이트

## 프로젝트 개요
- **브랜드명**: W2O SALADA (가칭, 변경 가능)
- **컨셉**: 젊고 건강한 샐러드 새벽배송 서비스
- **연관 사이트**: 다함푸드 (https://dahamfood.co.kr/) - 모회사
- **호스팅**: 카페24 (별도 도메인 예정)
- **기술 스택**: HTML5 / CSS3 / Vanilla JavaScript (WordPress 아님)

## 디자인 방향
- **톤**: 신선하고 역동적, 젊은 층 타겟
- **색상**: 그린 계열 기반, 활기찬 액센트 컬러
- **폰트**: Pretendard (한글) + 영문 산세리프
- **벤치마킹**: 컬리, 샐러디, 프레시코드

## 페이지 구성

### 1. 히어로 섹션
- 풀스크린 배경 영상 또는 고화질 이미지
- 타이핑 애니메이션 텍스트: "새벽, 신선함이 도착합니다"
- CTA 버튼 (주문하기)

### 2. 서비스 소개
- 스크롤 트리거 fade-in / slide-up 애니메이션
- 카드형 레이아웃으로 핵심 가치 전달

### 3. 메뉴 갤러리
- 호버 시 확대(scale) + 오버레이 효과
- 필터링 탭 (샐러드 / 그레인볼 / 프로틴 등)

### 4. 새벽배송 프로세스
- 타임라인 애니메이션 (주문 → 준비 → 배송 → 도착)
- 스크롤 연동 progress bar
- 카운트업 숫자 애니메이션 ("오늘 배송 1,234건")

### 5. 구독/주문 안내
- 구독 플랜 카드 (호버 시 강조)
- CTA 버튼

### 6. 고객 후기
- 자동 슬라이드 캐러셀
- 별점 + 텍스트 리뷰

### 7. 인스타그램 피드 (선택)
- SNS 연동 섹션

### 8. 푸터
- 회사 정보, 문의, SNS 링크

## 역동적 요소 (Dynamic Effects)

### 스크롤 애니메이션
- **Intersection Observer API** 사용
- 섹션 진입 시 fade-in, slide-up, slide-left/right
- 스크롤 위치에 따른 opacity/transform 변화
- stagger 애니메이션 (카드가 순차적으로 등장)

### 패럴랙스 효과
- 히어로 섹션 배경 이미지 패럴랙스
- 섹션 간 레이어 깊이감 표현
- CSS transform: translate3d() 활용 (GPU 가속)

### 호버 인터랙션
- 메뉴 카드: scale(1.05) + box-shadow 확대 + 오버레이 텍스트
- 버튼: 배경색 전환 + 아이콘 이동 애니메이션
- 네비게이션: 밑줄 슬라이드 효과

### 마이크로 애니메이션
- 로딩 시 로고 fade-in + 살짝 bounce
- 버튼 클릭 시 ripple 효과
- 아이콘 hover 시 회전 또는 pulse
- 스크롤 다운 화살표 bounce 반복

### 카운트업 애니메이션
- 숫자가 0에서 목표값까지 올라가는 효과
- 뷰포트 진입 시 트리거
- easeOutExpo 이징 함수 적용

### 타이핑 애니메이션
- 히어로 섹션 메인 카피 타이핑 효과
- 커서 깜빡임
- 여러 문구 순환 가능

### 풀스크린 영상 배경
- 히어로 섹션 autoplay, muted, loop 영상
- 오버레이 그라데이션으로 텍스트 가독성 확보
- 모바일에서는 정적 이미지로 대체 (성능)

### 부드러운 스크롤
- CSS scroll-behavior: smooth
- 네비게이션 클릭 시 해당 섹션으로 스무스 스크롤
- 스크롤 시 헤더 축소/색상 변경

### 슬라이드/캐러셀
- 고객 후기 자동 슬라이드 (일시정지 가능)
- 터치 스와이프 지원 (모바일)
- 무한 루프

### 페이지 전환 효과
- 첫 로딩 시 splash 애니메이션
- 섹션 간 부드러운 전환

## 성능 고려사항
- 애니메이션은 transform, opacity 위주 (리플로우 최소화)
- will-change 속성 적절히 사용
- 모바일에서는 복잡한 애니메이션 축소
- 이미지 lazy loading 적용
- prefers-reduced-motion 미디어 쿼리 존중
